## Installation

Install  chalk-advanced with npm

```bash
  npm install chalk-advanced
```
Install  chalk-advanceds with yarn
```bash
  yarn add  chalk-advanced
```
    
## Code Example

```js
const { black } = require("./src/chalk-advanced");

console.log(black("Hello World!"));
```
